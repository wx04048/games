Cómo finalizar tareas en segundo plano - Win 2000/XP - 98/ME

 

Win 2000/XP

 

Para obtener el mejor resultado, tienes que cerrar la mayoría de los programas de tu ordenador antes de instalar o ejecutar este programa.  

 

 

Para hacerlo, sigue los pasos siguientes:

 

 - Mantén pulsadas las teclas Ctrl y Alt del teclado.

 - Pulsa la tecla Supr una vez.

 

Aparecerá la ventana Seguridad de Windows.

Haz clic en el Administrador de tareas. Aparecerá la ventana del Administrador de tareas.

En esta pantalla hay tres fichas: Aplicaciones, Procesos y Rendimiento. En la pantalla de Aplicaciones, haz clic en cada una de las tareas y después haz clic en Finalizar tarea. Cuando hayas cerrado todas las tareas y la lista esté vacía, haz clic en la ficha Procesos.

En la pantalla Procesos, haz clic en cada uno de los elementos que no aparezcan en la lista siguiente y haz clic en Terminar proceso.  

  

NOTA: No hagas clic en Terminar proceso teniendo seleccionado alguno de los procesos que se enumeran a continuación.

 

System Idle Process

System

winlogon.exe

services.exe

svchost.exe

spoolsv.exe

csrss.exe

hidserv.exe

MSTask.exe

WinMgmt.exe

taskmgr.exe

IEXPLORE.EXE

Explorer.EXE

 

Cuando hayas cerrado todas las aplicaciones y procesos tal y como se ha descrito anteriormente, el sistema estará libre de programas en segundo plano que puedan causar problemas al juego.

 

 

Win 98/ME

 

Para obtener el mejor resultado, tienes que cerrar la mayoría de los programas de tu ordenador antes de instalar o ejecutar este programa.

 

 

Para hacerlo, sigue los pasos siguientes:

 

 - Mantén pulsadas las teclas Ctrl y Alt del teclado.

 - Pulsa la tecla Supr una vez.

 

Esto hará que aparezca la ventana Cerrar programa. Se deben dejar funcionando dos aplicaciones: "Explorer" y "Systray". Para finalizar todas las demás aplicaciones, haz lo siguiente:

 

 - Haz clic para resaltar la primera aplicación disponible.

 - Haz clic en "Finalizar tarea".

 

Esto hará que desaparezca la ventana Cerrar programa. Sigue los pasos anteriores para abrir de nuevo la ventana Cerrar programa y finalizar la siguiente aplicación disponible. Recuerda: no cierres ni EXPLORER ni SYSTRAY. Si lo haces, el sistema se volverá inestable y quizá tengas que reiniciar el ordenador. Si el ordenador se reinicia, tendrás que volver a repetir el proceso desde el principio.
