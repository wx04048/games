La instalación se bloquea y muestra un mensaje de error

 

Este problema puede tener multitud de causas, que dependen del mensaje de error que aparece. Si recibes un mensaje de error de transferencia de datos o de lectura de disco, es probable que el mismo CD/DVD sea el que esté causando el problema. Puedes probar a limpiarlo, a usar una unidad de CD/DVD diferente (si es que tienes más de una) o a instalar el juego en otro ordenador. Si aparece un mensaje tipo "fallo de página no válida", es recomendable cerrar los programas que se estén ejecutando al mismo tiempo que la instalación.

La instalación dura demasiado tiempo

 

Algunas instalaciones son bastante grandes, y tardan un rato en completarse. Antes de intentar solucionar este problema, asegúrate de que le has dado tiempo al ordenador para completar el proceso de instalación. Antes de hacer nada, busca signos de actividad en el ordenador. Fíjate en si las luces del disco duro o del CD/DVD-ROM parpadean y escucha si se oyen ruidos de actividad de estas unidades; sobre todo, fíjate en las barras de progreso de la instalación para ver si se mueven.
