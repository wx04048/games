Сообщения об ошибке

 

Появление заставки игры сопровождается выводом сообщения об ошибке.

 

Если вы пытаетесь запустить игру, а программа выдает сообщение об ошибке, попробуйте предпринять что-либо из нижеприведенного:

 

Завершите работу фоновых задач

Обновите драйвер видеокарты

 

 

 

Совет EA: если ни один из вышеперечисленных способов не помог вам устранить неисправность, обратите внимание на информацию, сопровождающую каждое сообщение об ошибке (кнопка Details), - обычно там можно обнаружить сведения о файле, вызвавшем ошибку. Иногда такой файл может относиться к программе, которую следует удалить, либо не запускать одновременно с игрой.
