Fel meddelande

 

Startskärmen visas och sedan får jag ett felmeddelande

 

Om ett felmeddelande visas efter att du har startat spelet kan du försöka lösa problemet genom att:  

 

 

Avsluta program som körs i bakgrunden

Uppdatera drivrutinen för grafikkortet

 

 

 

EA Help:  Om de här åtgärderna inte hjälper kan du se om det finns mer detaljerad information i felmeddelandet genom att klicka på knappen ”Detaljer”.  Ofta innehåller felmeddelandet namnet på den fil som orsakar problemet.  Ibland kan den här filen vara länkad till ett annat program i systemet som inte behövs eller som körs samtidigt som spelet. Om du söker efter filen i Utforskaren i Windows kan du kanske ta reda på vad filen används för. Om filen ingår i ett program som inte behövs kan du avinstallera eller stänga av programmet.
