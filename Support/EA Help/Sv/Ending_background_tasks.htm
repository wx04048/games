Avsluta program som körs i bakgrunden - Win 2000/XP - 98/ME

 

Win 2000/XP

 

 

Innan du installerar eller kör det här programmet bör du avsluta alla andra program som körs på datorn.  Gör så här:

 

 – Håll ned Ctrl- och Alt-tangenterna på tangentbordet.

 – Tryck en gång på Delete-tangenten.

 

Fönstret Windows-säkerhet visas.

Klicka på Aktivitetshanteraren.  Aktivitetshanteraren visas.

Det finns tre flikar i den här dialogrutan – Program, Processer och Prestanda.  Klicka på fliken Program, och avsluta alla program genom att markera programmet klicka på Avsluta aktivitet.  När du har avslutat alla program klickar du på fliken Processer.

Avsluta alla processer som inte finns i listan nedan genom att markera processen och klicka på Avsluta process.

  

Obs! Klicka inte på Avsluta process för någon av följande processer.

 

System Idle Process

System

winlogon.exe

services.exe

svchost.exe

spoolsv.exe

csrss.exe

hidserv.exe

MSTask.exe

WinMgmt.exe

taskmgr.exe

IEXPLORE.EXE

Explorer.EXE

 

När du har avslutat alla program och processer har du avslutat alla bakgrundsprogram som kan orsaka problem med spelet.

 

 

Win 98/ME

 

 

Innan du installerar eller kör det här programmet bör du avsluta alla andra program som körs på datorn.  Gör så här:

 

 – Håll ned Ctrl- och Alt-tangenterna på tangentbordet.

 – Tryck en gång på Delete-tangenten.

 

När du gör det visas fönstret Stäng program.  Du kan avsluta alla program utom två:  ”Explorer och ”Systray”.  Så här gör du för att avsluta ett program.

 

 – Klicka för att markera programmet.

 – Klicka på ”Avsluta aktivitet”

 

När du gör det stängs fönstret Avsluta program.  Upprepa proceduren ovan för att öppna fönstret Stäng program och avsluta nästa program.  Kom ihåg att du inte ska avsluta programmen EXPORER och SYSTRAY.  Om du gör det blir systemet instabilt, och eventuellt kan datorn startas om.  Om datorn startas om måste du upprepa den här proceduren från början.
