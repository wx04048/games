CD-felsökning - Win 2000/XP - 98/ME

Win 2000/XP

 

 

 

Ibland kan du bli uppmanad att sätta i en annan CD-skiva under installationen.  Kontrollera att den CD-skiva du sätter i verkligen är den som installationsprogrammet frågar efter.  Om det är rätt skiva kan du behöva vänta upp till en minut innan du kan klicka på OK för att fortsätta installationen.  Datorn kan behöva upp till en minut för att känna av vilken CD-skiva som du har satt in i CD-läsaren.  Om det ändå inte går att fortsätta installationen kan du kontrollera om datorn har hittat CD-skivan med ”Den här datorn”.  

 

Gör så här:

 

1.  Minimera installationsprogrammet genom att klicka på ikonen Minimera.  Du kan också minimera ett program genom att högerklicka på det i Aktivitetsfältet och sedan klicka på Minimera på snabbmenyn.  

2.  Klicka på Start-knappen och klicka på ”Den här datorn”.  

3.  Leta rätt på CD-läsaren.  

4.  Dubbelklicka på den enhetsbeteckning som motsvarar CD-läsaren.  Om du kan göra det i Windows betyder det att operativsystemet har hittat CD-skivan.  Maximera installationsprogrammet och klicka på OK för att fortsätta installationen.
