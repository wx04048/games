Konfigurera bildskärmen

 

 

Så här gör du för att konfigurera bildskärmen för spelet:

 

Dubbelklicka på Bildskärm i Kontrollpanelen och kontrollera att bildskärmen inte är inställd på VGA 640x480 256 färger. Om du använder den här inställningen kan du försöka ändra inställningen så att den stämmer med den bildskärm du använder. Du kan också försöka använda inställningen SVGA 800x600 (16-bitars färg).

 

Om du inte kan lösa problemet på det här sättet kan du försöka att uppdatera drivrutinen till grafikkortet från tillverkarens webbplats.
