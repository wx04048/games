'GET SETUP' Error When Installing DirectX

This error is typically due to a conflict in the system files when the DirectX installation cannot find a corresponding system file to continue the installation. You can, however, manually replace a file that should allow you to install DirectX.

 

  1. Insert the game disk into the drive.

  2. Click the Start button (Windows button for Windows Vista) on the Windows Taskbar.

  3. Select My Computer (Computer for Windows Vista).

  4. Right-click the game disk icon and select Explore.

  5. Double-click the DirectX folder.

  6. Right-click the 'dsetup.dll' file and select Copy.

  7. Close the window.

  8. Click the Start button (Windows button for Windows Vista) on the Windows Taskbar.

  9. Select My Computer (Computer for Windows Vista).

  10. Double-click the Local Disk C: drive.

  11. Double-click the WINDOWS folder.

  12. Double-click the SYSTEM32 folder.

  13. Right-click on an empty space in the folder and select Paste.

  14. Close all open windows.

  15. Now run the installation again.

 

If the issue persists, you should reinstall DirectX directly from Microsoft's web site.
