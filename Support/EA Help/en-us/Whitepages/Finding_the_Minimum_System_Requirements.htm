Finding the Minimum System Requirements

The minimum requirements for any Electronic Arts game can be found in multiple places.

 

The Game box

  1. Locate the System Requirements sticker.

    • The left side of the sticker lists all the hardware requirements.

    • The right side of the sticker lists the video requirements and the supported video chipsets.

 

The Readme file

The Readme is most easily found through the Start Menu location for the game. The location for these is different between EA Games and EA Sports titles, and if you chose a different location, you will need to go to that custom location instead. The System Requirements are typically the first section after the Table of Contents, but check the Table of Contents to verify its location.

 

  • EA Games default Readme location:

    1. Click the Start button (Windows button on Windows Vista) on the Windows Taskbar.

    2. Click All Programs.

    3. Click EA GAMES.

    4. Click the [game title].

    5. Click Readme.

  • EA Sports default Readme location:

    1. Click the Start button (Windows button on Windows Vista) on the Windows Taskbar.

    2. Click All Programs.

    3. Click EA SPORTS.

    4. Click the [game title].

    5. Click Readme.

 

The EAsy Info tool
The requirements listed here are the least complete, so try to use one of the other methods if possible. The only benefit of using the EAsy Info tool is that you get an instant comparison of your system and the requirements. However, it cannot usually give you full video card information and sometimes cannot fully detect all hardware configurations. The EAsy Info tool has a similar location to the Readme file; it's location is different between EA Games and EA Sports. If you chose a custom install location, please refer to that location instead.

 

  • EA Games default EAsy Info location:

    1. Click the Start button (Windows button on Windows Vista) on the Windows Taskbar.

    2. Click All Programs.

    3. Click EA GAMES.

    4. Click the [game title].

    5. Click EAsy Info.

  • EA Sports default EAsy Info location:

    1. Click the Start button (Windows button on Windows Vista) on the Windows Taskbar.

    2. Click All Programs.

    3. Click EA SPORTS.

    4. Click the [game title].

    5. Click EAsy Info.

  

The Game Requirements are listed on the right side of the window. The only System Requirements information listed here is the Operating System, necessary system memory (RAM), CPU speed, DirectX version, and display memory. For further information, you must use one of the other methods for checking the System Requirements.

 

The Support site

  1.  Go to our support site at http://support.ea.com.

  2. Click Technical Support.

  3. In the Platform drop-down menu, choose PC.

  4. Choose your game name from the Product menu.

  5. In the search text box, type requirements.

  6. Click the Search button.

  7. A list of related answers will appear. Click the question in the Summary field to view the answer. The question in this case would be 'What are the system requirements?'

  8. This answer is most useful as a reference to check your system before purchasing the game.
