Crash Troubleshooting

A crash is when a program shuts down unexpectedly, usually returning you back at your Windows Desktop. A game will typically crash as such when there are conflicts between its files or requirements and the hardware in the computer it is trying to run on. The following section of the help troubleshooting manual will guide you through common issues that can cause crashes while trying to play. There are two common types of crashes which point to different causes. Click the type of crash you are experiencing to learn how to troubleshoot it.
