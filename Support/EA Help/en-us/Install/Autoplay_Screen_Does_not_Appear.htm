Autoplay Screen Does not Appear

When the Autoplay screen does not appear, it is usually due to processes running in the background that conflict with the game.

 

First, make sure the game disk is in the drive. If you have a DVD edition disk, please make sure you are using a DVD-ROM drive and that all of your background tasks have been shut down.

How to end background tasks

Once your background tasks have been ended you can start the installation manually. To manually start the installation process:

  1. Click on the Start button (Windows button for Windows Vista) on the Windows Taskbar.

  2. Select My Computer (Computer for Windows Vista).

  3. Right-click the game disk icon and select Open.

  4. Double-click the Setup icon to start the installation process.

 

If you have already installed your game and want to play:

  1. Find the Game Icon on your Desktop.

  2. If you are unable to find an icon for your game on the Desktop, click the Start button (Windows button for Windows Vista) on the Windows Taskbar.

  3. Select All Programs.

  4. Find the folder for your game (usually located under the EA Games folder heading) and select the icon for the game you are trying to play.
