4 Digit Error Code During Installation

If you are experiencing a four-digit error code while installing or clicking to play the game, please ensure that you have tried the following troubleshooting steps.

  • Verify that you have the correct disk in the drive. If the incorrect CD is in the drive, the needed files will not be found and this error will appear. You can usually tell which disk is needed by either a picture or text reference to the needed disk.

  • End all unnecessary background processes. These may conflict with the game and use system resources that are needed for the game to run properly.
    How to end background tasks

  • Try troubleshooting the CD/DVD-ROM drive. These steps will help resolve any conflicts reading the disk itself.
    How to troubleshoot the CD/DVD-ROM

 

Further issues may indicate a possible issue with SecuROM. The most efficient way for us to assist is to have you produce an analysis file for evaluation. In order to generate this analysis file, please do the following:

  1. Make sure the original 'play' disc is in a physical, local CD or DVD drive.

  2. Restart your computer (the reboot is to assure that there are no additional instances of the software running when generating the file).

  3. Right-click on the application's Desktop shortcut, or the actual application's executable file in the installation folder, and a menu will appear.

  4. Select Launch Analysis from the menu.

  5. A disclaimer window may appear on your screen. To proceed from there, click Yes. Another window may appear stating that an analysis log will be generated; if so, click OK. If any error messages appear, click Cancel to continue.

  6. After a period of time, the program will generate a file named AnalysisLog.sr0 and place it at the root of your hard drive (usually C:). When the file has been generated, a dialog may appear stating it has finished. Click OK to complete the process. To locate it, go to My Computer and open your hard drive (usually C:).

    • Note: Microsoft Vista users should refer to the further instructions below.

  7. Please compress the file using a program such as WinZip, WinRAR, or WinAce (to avoid email corruption), and attach it to your email to support@securom.com.

 

For Microsoft Vista Users:

Follow the steps outlined above through step 6. Since you're using Vista, the file will be stored in the following path:

C:\Users\[YOUR  USER]\AppData\Local\Temp\AnalysisLog.sr0.

  • Note: If by chance this folder is hidden, you may have to display hidden folders. To do so, please follow these steps:

    1. Click the Windows button on the Windows Taskbar.

    2. Select Control Panel.

    3. Click Appearance and Personalization and select Folder Options.

    4. Click the View tab, then select Show hidden files and folders. (You should now have access to the folder.)

 

If you do not see the Launch Analysis option available in the menu after you re-start your computer, you can generate it using the command line. Please follow these steps:

  1. Make sure the original 'play' disc is in a physical, local CD, or DVD drive.

  2. Restart your computer. (The reboot will ensure that there are no additional instances of the software running when generating the Analysis file).

  3. Right-click the application's Desktop shortcut and a menu will appear and select Properties.

  4. Highlight and copy everything in the Target window.

  5. Click the Windows button on the Windows Taskbar.

  6. Click All Programs.

  7. Click Accessories.

  8. Click Run.

  9. Type cmd in the text box and press the Enter key. (A 'DOS' window will appear.)

  10. Type cd\ and press the Enter key to get to the root of your hard drive. (You may have to type cd.. several times to get to C:\).

  11. Right-click on an empty area inside the 'DOS' window and select Paste.

  12. Press the Spacebar on your keyboard once, add the text /secuexp and press the Enter key.

  13. A disclaimer window may appear on your screen. To proceed from there, click Yes. Another window may appear stating that an analysis log will be generated; if so, click OK. If any error messages appear, click Cancel to continue.

  14. After a period of time, the program will generate a file named AnalysisLog.sr0 and place it at the root of your hard drive (usually C:). When the file has been generated, a dialog may appear stating it has finished. Click OK to complete the process. To locate it go to My Computer and open your hard drive (usually C:).

  15. Please compress the file using a program such as WinZip, WinRAR, or WinAce (to avoid email corruption), and attach it to your email to support@securom.com.
