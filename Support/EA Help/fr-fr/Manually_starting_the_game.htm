Démarrer le jeu manuellement  - Win 2000/XP - 98/ME

 

Win 2000/XP

 

 

Comment trouver votre jeu Electronic Arts dans le menu "Démarrer" ?

 

1.  Cliquez sur le bouton "Démarrer".

2.  Cliquez sur "Programmes".

3.  La plupart des jeux apparaissent dans les répertoires suivants : "EAGAMES", "EASPORTS" ou "Electronic Arts". Si l'un de ces répertoires apparaît, cliquez sur son icône puis cliquez sur l'icône du jeu pour le lancer.

 

Si ces répertoires n'apparaissent pas dans votre menu "Démarrer", vous pouvez lancer le jeu en accédant directement au CD/DVD :

 

1.  Faites un double clic sur l'icône Poste de travail située sur votre Bureau.

2.  Faites un clic droit sur le lecteur dans lequel vous avez inséré votre CD/DVD.

3.  Choisissez "Explorer".

 

Dans la fenêtre suivante, vous devriez voir l'icône Setup ou Autorun. Faites un double clic sur ce fichier.
