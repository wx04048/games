Fermer les applications d'arrière-plan - Win 2000/XP - 98/ME

 

Win 2000/XP

 

Pour obtenir des performances optimales, vous devrez fermer la plupart des programmes en cours sur votre ordinateur avant d'installer ou de démarrer le programme.

 

 

Voici comment procéder :

 

 - Maintenez les touches Ctrl et Alt enfoncées.

 - Appuyez une fois sur la touche Suppr.

 

Sous Windows 2000

 

La fenêtre "Gestionnaire des tâches de Windows" apparaît.

Cet écran comporte trois onglets : "Applications", "Processus" et "Performances". Dans l'onglet "Applications", sélectionnez tous les éléments de la liste puis cliquez sur le bouton "Fin de tâche". Lorsque vous avez fermé tous les programmes de la liste, cliquez sur l'onglet "Processus".

 

Dans l'onglet "Processus", cliquez sur chacun des noms des processus SAUF ceux qui apparaissent dans la liste ci-dessous et cliquez sur le bouton "Terminer le processus".

 

Processus inactif du système

System

winlogon.exe

services.exe

svchost.exe

spoolsv.exe

csrss.exe

mstask.exe

winmgmt.exe

taskmgr.exe

explorer.exe

smss.exe

lsass.exe

 

Sous Windows XP

 

La fenêtre "Gestionnaire des tâches de Windows" apparaît.

Cet écran comporte quatre onglets : "Applications", "Processus", "Performances" et "Mise en réseau". Dans l'onglet "Applications", sélectionnez tous les éléments de la liste puis cliquez sur le bouton "Fin de tâche". Lorsque vous avez fermé tous les programmes de la liste, cliquez sur l'onglet "Processus".

Dans l'onglet "Processus", fermez tous les processus associés au nom de l'administateur dans la colonne "Nom de l'utilisateur", à l'exception d'explorer.exe et de taskmgr.exe, puis cliquez sur le bouton "Terminer le processus".

 

Remarque : veillez à ne pas terminer les processus correspondants à SYSTEM, SERVICE LOCAL et SERVICE RESEAU dans la colonne "Nom de l'utilisateur".

 

Cette procédure vous permet de fermer tous les programmes d'arrière-plan qui pourraient poser des problèmes lors de l'utilisation de votre jeu.

 

 

Win 98/ME

 

Pour obtenir des performances optimales, vous devrez fermer la plupart des programmes en cours sur votre ordinateur avant d'installer ou de démarrer le programme. Voici comment procéder :

 

 

Voici comment procéder :

 

 - Maintenez les touches Ctrl et Alt enfoncées.

 - Appuyez une fois sur la touche Suppr.

 

La fenêtre "Fermer le programme" apparaît. Fermez toutes les applications sauf Explorer et Systray. Pour fermer toutes les autres applications, voici la procédure à suivre :

 

 - Faites un clic gauche sur la ligne de l'application concernée pour qu'elle soit surlignée.

 - Cliquez sur le bouton "Fin de tâche".

 

La fenêtre "Fermer le programme" disparaît. Ouvrez à nouveau la fenêtre "Fermer le programme" en suivant la procédure ci-dessus puis fermez l'application suivante. Vous ne devez pas fermer les applications Explorer ou Systray. Si vous le faites, cela pourrait entraîner l'instabilité de votre système ou faire redémarrer votre ordinateur. Si votre ordinateur redémarre, vous devrez recommencer cette procédure depuis le début.
