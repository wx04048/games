Supprimer les fichiers temporaires

 

Windows garde parfois en mémoire des fichiers temporaires qui ne sont pas nécessaires et qui peuvent altérer le bon fonctionnement d'un programme. Pour supprimer ces fichiers, suivez la procédure ci-dessous :

 

 

1.  Faites un double clic sur l'icône Poste de travail.

2.  Faites un double clic sur le disque local ou sur le lecteur C:\.

3.  Faites un double clic sur le répertoire "Windows".

4.  Si la fenêtre "Windows Avertissement" s'affiche, cliquez sur la ligne "Afficher les fichiers".

5.  Faites un double clic sur le répertoire "Temp".

6.  Cliquez sur "Edition" dans la barre de menu.

7.  Cliquez sur "Sélectionner tout".

8.  Cliquez sur "Fichier".

9.  Cliquez sur "Supprimer".
