Espace libre sur le disque dur

 

S'assurer qu'il y a suffisamment d'espace libre sur le disque dur

 

 

Voici comment procéder pour savoir quel est l'espace libre disponible sur votre disque dur :

 

1.  Faites un double clic sur Poste de travail.

2.  Faites un clic droit sur le lecteur sur lequel vous souhaitez installer le jeu (C:\ en général).

3.  Faites ensuite un clic gauche sur "Propriétés".

 

Le panneau des propriétés s'affiche pour le lecteur sélectionné. Ce panneau vous permet de consulter l'espace libre et l'espace utilisé sur le disque dur. Si l'espace libre total n'est pas supérieur d'au moins 100 Mo au volume minimum nécessaire au fonctionnement du jeu, il est possible que vous rencontriez des problèmes lors de l'installation ou de l'utilisation du jeu.

 

Voici comment procéder pour libérer de l'espace sur votre disque dur :

 

1.  Faites un double clic sur Poste de travail.

2.  Faites un clic droit sur le lecteur sur lequel vous souhaitez installer le jeu (C:\ en général).

3.  Faites un clic gauche sur "Propriétés".

 

Le panneau des propriétés s'affiche pour le lecteur sélectionné. Appuyez sur le bouton "Nettoyage de disque". Un panneau apparaît alors avec la liste des fichiers que vous pouvez supprimer sans problème. Vous pouvez supprimer ces fichiers depuis ce panneau.
