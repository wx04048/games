L'écran d'introduction s'affiche, puis il est suivi d'un message d'erreur.

 

Si vous recevez un message d'erreur après avoir tenté de lancer le jeu, essayez les procédures décrites ci-dessous.

EA Assister:  Si le problème subsiste, cliquez sur "Détails" dans la fenêtre du message d'erreur. Il y est souvent fait référence au fichier à l'origine du problème. Ce fichier est parfois lié à un autre programme du système qui n'a pas besoin d'être installé ou de fonctionner pendant que vous jouez. Vous pouvez effectuer une recherche sur ce fichier pour en découvrir la nature. Vous pourrez alors peut-être désactiver ou désinstaller ce programme.
