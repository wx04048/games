Odstraňování potíží s disky CD nebo DVD

Systém Windows 2000/XP

 

Instalační program někdy požádá o předchozí nebo další disk CD nebo DVD. Ujistěte se, že disk CD nebo DVD, který jste vložili do jednotky, je skutečně diskem, který instalační program vyžaduje. Pokud je, je někdy nutné počkat až minutu a teprve potom pokračovat v instalaci stisknutím tlačítka OK.

 

Počítač někdy potřebuje přibližně minutu, aby zaregistroval, že je v jednotce vložen příslušný disk CD nebo DVD. Nebude-li disk stále nalezen, zkuste minimalizovat instalační program a pomocí okna Tento počítač zkontrolovat, zda byl disk rozpoznán.  

 

 

 

Postup:

 

1. Minimalizujte instalační program klepnutím na ikonu Minimalizovat. Instalační program také můžete minimalizovat klepnutím pravým tlačítkem myši na jeho tlačítko na hlavním panelu a výběrem příkazu Minimalizovat.  

2. Klepněte na tlačítko Start a potom na složku Tento počítač.  

3. Vyhledejte jednotku CD-ROM nebo DVD-ROM.  

4. Poklepejte na písmeno jednotky. Pokud vám to systém Windows umožní, znamená to, že operační systém rozpoznal disk CD nebo DVD. Maximalizujte instalační program a pokračujte v instalaci klepnutím na tlačítko OK.
