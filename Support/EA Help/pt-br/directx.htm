DirectX

 

O que é o DirectX?   

 

O DirectX é um utilitário desenvolvido pela Microsoft que permite que o seu software, hardware e computador “falem uma mesma língua”.  O DirectX é uma atualização para o seu Sistema Operacional que permite a utilização das imagens e sons mais eficientes.  Como a maioria dos programas de software para Windows, o DirectX deve ser instalado em seu computador.  

 

Se você tiver problemas para instalar o DirectX em seu computador, tente o seguinte:

 

As duas causas mais comuns de uma interrupção na instalação quando o DirectX tenta se inicializar é a falta de espaço em disco e problemas com os seus drivers de vídeo e som.  Enquanto que o DirectX não ocupa muito espaço no disco rígido, você precisa compensar o fato de que o Windows está usando algum espaço no disco rígido para amemória virtual (um arquivo de troca).
